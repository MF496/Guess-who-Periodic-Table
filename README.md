# Guess-who-Periodic-Table
A guess who spinoff with the periodic table, local hosted and 2 player

Node JS is needend to run this project. Additionaly, need to download packet express, socket.io and charcoal.
**Index.html has to be moved into sub-folder public to work**

How to start the project
1. Navigate to the directory with cd
2. Run ``node server.js``
3. In the console the localhost adress and others player (your local ip) are listed.
4. Thats it!
